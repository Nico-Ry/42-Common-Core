# fract-ol

fract-ol is a graphics project from the 42 Common Core.

The goal is to render mathematical fractals in real time using C and MiniLibX, with interactive zoom, movement and color controls.

This project connects C programming, graphics, complex numbers and visual experimentation.

---

## Overview

The program opens a graphical window and renders a selected fractal.

Supported fractals:

* Mandelbrot
* Julia
* Burning Ship

Each fractal is generated by iterating mathematical formulas over points on the complex plane and coloring pixels depending on how quickly the point escapes.

```text id="fr6at1"
screen pixel
  ↓
map to complex plane
  ↓
iterate fractal formula
  ↓
calculate escape value
  ↓
choose color
  ↓
draw pixel
```

---

## Supported Fractals

### Mandelbrot

The Mandelbrot set is generated by repeatedly applying a formula to each point of the complex plane.

It is one of the most famous fractals and produces detailed recursive shapes when zooming in.

### Julia

The Julia set uses a similar formula to Mandelbrot, but with a fixed complex parameter.

In this project, Julia can be explored interactively and locked with a key once an interesting shape is found.

### Burning Ship

The Burning Ship fractal is a variation that uses absolute values in the iteration formula.

It creates sharp, flame-like structures and is visually different from Mandelbrot and Julia.

---

## Features

* Render Mandelbrot, Julia and Burning Ship fractals
* Zoom in and out with the mouse wheel
* Move the viewpoint with arrow keys
* Switch fractals with keyboard shortcuts
* Change color palettes with keyboard keys
* Lock Julia interaction
* Reset the fractal view
* Close the window with ESC
* Display a help/manual message from the terminal

---

## Controls

| Input           | Action                            |
| --------------- | --------------------------------- |
| `ESC`           | Close the window                  |
| `1` / `2` / `3` | Switch between available fractals |
| `Q` to `Y`      | Change fractal colors             |
| `A` to `H`      | Change fractal colors             |
| Mouse scroll    | Zoom in and out                   |
| Arrow keys      | Move the viewpoint                |
| `L`             | Lock Julia fractal interaction    |
| `0`             | Reset the fractal                 |

---

## Usage

```bash id="s0dfxo"
./fractol mandelbrot
./fractol julia
./fractol burning_ship
```

If the arguments are missing or invalid, the program displays a help menu explaining usage and controls.
