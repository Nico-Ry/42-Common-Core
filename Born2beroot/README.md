# Born2beroot

Born2beroot is a system administration project from the 42 Common Core.

The goal was to configure a secure Linux virtual machine following strict rules around users, groups, passwords, SSH, firewall configuration, sudo policies and system monitoring.

No virtual machine image is included in this repository.

---

## Overview

The original project was evaluated through a configured virtual machine.

This repository keeps documentation only, because committing a full VM image would be unnecessary and impractical.

The project focused on installing, configuring and securing a small Linux server environment.

## Monitoring Script

A monitoring script was used to display system information at regular intervals.

Typical information included:

* system architecture
* CPU information
* memory usage
* disk usage
* CPU load
* last boot time
* LVM status
* active connections
* user count
* IP and MAC address
* sudo command count

This helped connect Linux commands, system files and scheduled automation.
