# Born2beroot
Born2beroot is a system administration project from the 42 Common Core.

The goal was to configure a secure virtual machine following strict rules around users, groups, passwords, SSH, firewall configuration, sudo policies and system monitoring.

No VM image is included in this repository.

## Main topics

- Linux system administration
- virtual machines
- Debian installation
- SSH configuration
- UFW firewall
- user and group management
- password policy
- sudo configuration
- cron jobs
- monitoring script
- system hardening basics

## What I learned

This project introduced me to Linux server configuration, security rules, service management and basic infrastructure administration.

## Notes

The original project was evaluated through a configured virtual machine, so this folder only contains documentation and optional sanitized screenshots.
