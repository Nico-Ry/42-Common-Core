# MiniRT

MiniRT is a small ray tracing engine written in C.

The goal of the project is to render simple 3D scenes using ray tracing principles.

## Overview

The program parses a scene file, builds the corresponding world representation, casts rays from a camera into the scene, computes intersections with objects and renders the result using lighting calculations.

- 3D geometry
- Vector math
- Camera systems
- Object intersections
- Lighting
- Shadows

- MiniLibX graphics
