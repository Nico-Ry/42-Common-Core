# ft_transcendence

ft_transcendence is the final project of the 42 Common Core.

This folder documents the project as part of my Common Core repository.

The complete public version with detailed documentation is available here:

https://github.com/Nico-Ry/Ft_transcendence-personal

## Overview

The project is a full-stack multiplayer web application with authentication, realtime features, user profiles, friends, chat, game history, achievements and multilingual support.

## My main contributions

- Backend authentication architecture
- JWT authentication with HTTP-only cookies
- 42 OAuth integration
- TOTP-based two-factor authentication
- Friends system API
- Realtime friends and presence updates
- Swagger/OpenAPI documentation
- Backend error handling and API protection

## Tech stack

- NestJS
- TypeScript
- PostgreSQL
- Prisma
- Docker
- Nginx
- Socket.IO
- React
- JWT
- OAuth2
- 2FA
